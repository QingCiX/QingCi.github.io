搭建博客(一)——环境安装

安装 Git

  1. 访问 git 官方地址，下载对应安装包（本记录以 Windows10 为对象）

  2. 验证安装

  3. 配置账号

配置 SSH key

  1. 生成 ssh key

    Note：邮件地址是 GitHub 账号绑定的邮件地址，生成的文件会自动保存到 C:/用户/Administrator/.ssh/ 下

  2. 为 GitHub 账号添加 SSH key
    添加方法：“Settings” -> “SSH and GPG keys” -> “New SSH key” -> 将上一步生成的文件 id_rsa.pub 中的内容全部拷贝到“Key”中 -> “Add SSH key”

  3. 在 C:/用户/Administrator/.ssh/ 下新建文件 config，内容如下

  4. 测试配置

    Note：这一步要切换网络，不要使用校园网，会被屏蔽

安装 Nodejs

  1. 访问 node.js 官方地址，下载对应安装包

  2. 验证安装

安装 Hexo

  1. 访问 Hexo 官方文档，按照教程配置

  2. 验证安装

  3. 新建博客

    Note：这一步比较慢，不是网络问题，可能是源的问题

  4. 生成静态网页

    Note：生成的静态网页只能在本地预览，在浏览器输入 http://localhost:4000/ 查看

  5. 新建 GitHub 仓库
    新建仓库，命名为“账户名.github.io”，设置为“Public”，选择“Add a README file”，点击“Create repository”

  6. 安装插件

    Note：这一步在第3步生成的 [folder] 下完成，不安装这个插件无法将博客部署至线上

  7. 编辑 _config.yml

    Note：这个文件在在第3步生成的 [folder] 下

  8. 部署到 Github 上

    Note：如果使用的是校园网则需要切换网络，否则会被屏蔽

  9. 访问个人博客
    在浏览器输入 https://你的账户名.github.io/
